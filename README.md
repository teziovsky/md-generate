## Get started 🔥

> Open the root (where package.json is) directory of your project and:

### Install as a global package and run

```bash
  npm install -g md-generate
  # or
  pnpm add -g md-generate
  # then
  md-generate
```

### If you have npx installed

```bash
  npx md-generate
  # or
  pnpx md-generate
```
